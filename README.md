# wordle-solver
Program to solve the wordle puzzle

## Running

### Cloning the repository

```sh
git clone https://github.com/animesh-chouhan/wordle-solver.git
cd wordle-solver
```

### Install requirements

```sh
pip install -r requirements.txt
```

### Run

```
python3 main.py 2> log.txt 
```
